# Exercícios da Universidade

Este repositório contém exercícios realizados para a licenciatura.

## Conteúdo

Os materiais incluídos dizem respeito às seguintes áreas:

- Arquitetura de Computadores  
- Estruturas de Dados e Algoritmos

## Objetivo

Organizar e acompanhar a evolução dos exercícios práticos desenvolvidos ao longo das aulas.

## Estrutura

Os ficheiros estão organizados por aulas ou temas.
